## 📋 Sobre

Vates Vesperion é um site de leituras oraculares, tarot, baralho cigano e aconselhamento místico. Descubra mensagens, orientações e reflexões através de consultas personalizadas, com foco em autoconhecimento e espiritualidade.

**25 Serviços Implementados:**
- 🔮 **Respostas Rápidas** - Pergunta Objetiva e Detalhada (R$ 5,99-9,99)
- 📖 **Consultas Completas** - 30/45/60 minutos (R$ 64,99-99,99)
- 💕 **Leituras de Amor** - 7 tiragens especializadas (R$ 29,99-54,99)
- 💼 **Financeiro & Carreira** - 5 tiragens (R$ 29,99-49,99)
- 📅 **Planejamento** - Bússola Semanal/Mensal (R$ 19,99-34,99)
- ⭐ **Direcionamento & Aprofundamento** - Mandala da Vida, Cruz Celta, Ferradura
- 🃏 **4 Oráculos** - Tarot, Baralho Cigano, Sibilla, Cleromancia

Acesse em [vesperion.com.br](https://vesperion.com.br).

## 🛠️ Instalação e Uso

```bash
# Clone o repositório
git clone https://github.com/paulemacedo/vates-vesperion.git

# Entre no diretório
cd vates-vesperion

# Instale as dependências
npm install

# Execute em modo de desenvolvimento
npm run dev

# Build para produção
npm run build
```
